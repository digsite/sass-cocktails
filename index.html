Vertical/Horizontal Centering

Horizontally/vertically align element to the center of its wider/taller container. Mixin assumes that both the parent has a defined width & height property and has position: relative;.

Original code and article found on CSS-Tricks

.both { @include centerer(); }
.horizontal { @include centerer(true, false); }
.vertical { @include centerer(false, true); }
I'm centered on both axis!
I'm centered horizontally!
I'm centered vertically!
